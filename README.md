# 🎬 CineClássicos - ATV02-DW-REACT

Projeto desenvolvido para a disciplina de Desenvolvimento Web III - Fatec Registro.

## 📋 Sobre o Projeto

Aplicação web que consome a API do TMDb (The Movie Database) para exibir filmes clássicos das décadas de 60, 70 e 80.

## 🚀 Tecnologias Utilizadas

- **Frontend:** React, Next.js, CSS Modules
- **Backend:** Node.js, Express
- **API:** TMDb (The Movie Database)

## 📦 Como Executar

### Pré-requisitos
- Node.js instalado
- Conta no [TMDb](https://www.themoviedb.org/signup) para obter uma API Key

### Backend
```bash
cd api
npm install
# Coloque sua API Key no arquivo server-movies.js
node server-movies.js
